# 📊 E-Commerce Sales Analysis Using MySQL & Power BI

## 📌 Project Overview

This project focuses on analyzing e-commerce sales data using **MySQL** and **Power BI** to generate meaningful business insights related to:

* 💰 Sales Performance
* 📈 Profitability Analysis
* 👥 Customer Segmentation
* 🌍 Regional Sales Trends
* 🛒 Product Performance
* 💳 Payment Method Analysis
* 🚚 Delivery Status Tracking

The project demonstrates an end-to-end **Data Analytics workflow** including:

✅ Data Import

✅ SQL Querying

✅ KPI Analysis

✅ Business Insight Generation

✅ Dashboard Visualization

# 📌 Key Business Insights

📈 Total Sales exceeded **9.3M+**

💰 Total Profit exceeded **2.75M+**

🛒 Grocery category generated highest sales.

🌍 North region achieved highest revenue contribution.

💳 Debit Card emerged as the most preferred payment method.

👥 Home Office segment generated highest sales contribution.

🍚 Organic Rice became the top-performing product.
